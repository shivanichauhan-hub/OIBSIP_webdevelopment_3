 Temperature Converter
A simple and user-friendly web application that converts temperatures between Celsius, Fahrenheit, and Kelvin.
 Objective
Create an interactive temperature converter that allows users to:

Input a temperature value in any unit (Celsius, Fahrenheit, or Kelvin)
Select which unit they are converting from
Get instant conversion results for all three temperature units
Enjoy a clean and responsive user interface

 Tools & Technologies Used

HTML5 - Structure and content
CSS3 - Styling and visual design
JavaScript - Conversion logic and interactivity

 Features

 Input validation to ensure only numbers are accepted
 Radio buttons for easy unit selection
 One-click conversion with a dedicated button
 Real-time error handling
 Display all three temperature conversions simultaneously
 Keyboard support (press Enter to convert)
 Responsive design that works on all devices
 Beautiful gradient background

How It Works

User enters a temperature value in the input field
User selects the input unit (Celsius, Fahrenheit, or Kelvin) using radio buttons
User clicks the "Convert" button or presses Enter
The app validates the input and shows an error if invalid
Results are displayed showing conversions to all three units

Conversion Formulas
The app uses these standard conversion formulas:
Celsius to Fahrenheit: (C × 9/5) + 32
Celsius to Kelvin: C + 273.15
Fahrenheit to Celsius: (F - 32) × 5/9
Kelvin to Celsius: K - 273.15

How to Use
Download or clone this repository
Open the temperature-converter.html file in any web browser
Enter a temperature value
Select your input unit
Click "Convert" to see the results!
